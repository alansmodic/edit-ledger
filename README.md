# Edit Ledger

Editorial-friendly revision history for WordPress with visual diffs and preview capabilities.

## Description

Edit Ledger replaces WordPress's basic revision slider with a visual timeline showing who changed what and when, with better diff visualization. Perfect for editorial teams who need to track content changes.

## Features

- **Block Editor Sidebar Timeline**: Visual timeline of revision cards showing author, timestamp, and change type
- **Visual Diff Viewer**: Toggle between inline and side-by-side views with green/red highlighting for additions/deletions
- **Media Change Tracking**: See which images, videos, and embeds were added or removed
- **Admin Dashboard**: Overview of recent revisions across all posts with filtering
- **Restore Functionality**: Easily restore any previous revision with one click
- **Clean Text Diffs**: Strips HTML to show editors clean, readable text changes

## Requirements

- WordPress 6.4 or higher
- PHP 7.4 or higher

## Installation

1. Upload the `edit-ledger` folder to `/wp-content/plugins/`
2. Activate the plugin through the WordPress admin
3. Open any post in the block editor and click "Edit Ledger" in the sidebar

## Usage

### Block Editor
- Open any post/page in the block editor
- Click the clock icon or "Edit Ledger" in the sidebar menu
- Click any revision card to expand and see what changed
- Use "View Diff" to see detailed changes in a modal
- Use "Restore This Version" to revert to a previous revision

### Admin Dashboard
- Navigate to "Edit Ledger" in the WordPress admin menu
- Filter revisions by author or date range
- Click "View Diff" on any revision to see changes
